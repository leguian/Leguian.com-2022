## Getting Started
Install dependencies
```bash
npm install
```

### Use in development
```bash
npm start
```

### Build for production
```bash
npm run build
```

### For debugging purposes
```bash
npm run debug
```

## Configuration
To change the title, description, author data, menu/nav item, etc, go to `src/_data/`.
